FacturaAI API

API para OCR + IA de facturas argentinas - Extracción automática de datos estructurados.

Endpoints

GET /health

Health check - Verificar estado del servicio

POST /v1/process

Subir imagen de factura para procesar

GET /v1/jobs/{job_id}

Obtener estado y resultados del job

GET /v1/jobs/{job_id}/export

Exportar datos como JSON

Ejemplo de Uso

# Subir factura
curl -X POST -F "file=@factura.png" http://localhost:8000/v1/process

# Obtener resultados
curl http://localhost:8000/v1/jobs/{job_id}

Campos Extraídos

  • Número de factura
  • Fecha de emisión
  • Fecha de vencimiento
  • CUIT del vendedor y cliente
  • Nombre y domicilio
  • Subtotal, IVA y total
  • Items de la factura
